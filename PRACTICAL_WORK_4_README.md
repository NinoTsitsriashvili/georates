## გამოყენება

პროგრამის გაშვების შემდეგ, თქვენ ნახავთ მენიუს, საიდანაც შეგიძლიათ აირჩიოთ სასურველი ვარჯანტი:

- **0** - გამოსვლა
- **1** - მაგალითი პროგრამა 1
- **2** - 20+ გამოსახულება სტანდარტული ფუნქციებით
- **3-39** - სხვადასხვა გამოთვლის ვარჯანტები

## გამოყენებული სტანდარტული ფუნქციები

- `sqrt(x)` - კვადრატული ფესვი
- `exp(x)` - ექსპონენციალური ფუნქცია
- `log(x)` - ნატურალური ლოგარითმი
- `log10(x)` - ათობითი ლოგარითმი
- `fabs(x)` - აბსოლუტური მნიშვნელობა
- `ceil(x)` - დამრგვალება ზემოთ
- `floor(x)` - დამრგვალება ქვემოთ
- `pow(x, y)` - x ხარისხად y
- `sin(x)` - სინუსი (რადიანებში)
- `cos(x)` - კოსინუსი (რადიანებში)
- `tan(x)` - ტანგენსი (რადიანებში)
- `atan(x)` - არქტანგენსი (რადიანებში)
